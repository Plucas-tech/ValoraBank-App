# 💳 Valora Bank

Aplicativo bancário mobile desenvolvido com React Native, Expo e Firebase.

O projeto simula um banco digital moderno, permitindo autenticação de usuários, saldo individual, extrato de movimentações e navegação entre telas.

---

# 🚀 Tecnologias utilizadas

- React Native
- Expo
- Firebase Authentication
- Firebase Firestore
- React Navigation
- Expo Vector Icons

---

# 📱 Funcionalidades

✅ Login de usuários  
✅ Cadastro de conta  
✅ Recuperação de senha  
✅ Saldo personalizado por usuário  
✅ Extrato de movimentações  
✅ Navegação com Bottom Tabs  
✅ Interface moderna inspirada em bancos digitais
